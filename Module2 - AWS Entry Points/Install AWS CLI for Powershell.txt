
Install AWS CLI for powershell
*	Install-Module -Name AWS.Tools.Installer

Install tools for every service you need
*	Install-AWSToolsModule AWS.Tools.EC2,AWS.Tools.S3 -CleanUp

Run Commands
*	Get-S3Bucket

Ref: https://docs.aws.amazon.com/powershell/latest/userguide/pstools-getting-set-up-windows.html#ps-installing-awswindowspowershell
